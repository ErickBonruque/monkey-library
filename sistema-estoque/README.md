# Sistema de Estoque

Aplicação web de gerenciamento de estoque desenvolvida com Next.js 16, React 19 e TypeScript.

## Funcionalidades

- Página inicial com apresentação do sistema
- Páginas públicas: Sobre e Contato
- Autenticação com controle de acesso à área administrativa
- Dashboard com métricas e gráficos do estoque
- Visualização, cadastro e baixas de produtos
- Gerenciamento de usuários

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
