# Atividade Next.js - Sistema de Usuários

Aplicação simples em Next.js (App Router) usando `useState`, `useEffect` e roteamento.

## Rotas

- `/` - Home
- `/usuarios` - Lista de usuários (mockados)
- `/usuarios/[id]` - Detalhes do usuário
- `/sobre` - Página simples

## Como rodar

```bash
npm install
npm run dev
```

Abra http://localhost:3000
