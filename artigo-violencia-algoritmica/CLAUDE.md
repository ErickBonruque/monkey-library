# CLAUDE.md — Violência Algorítmica em Cidades Inteligentes

## GSD Workflow

This project uses GSD (Get Shit Done) for phase-based execution with verification.

### Project State

- **Project file:** `.planning/PROJECT.md`
- **Requirements:** `.planning/REQUIREMENTS.md`
- **Roadmap:** `.planning/ROADMAP.md`
- **Current phase tracking:** `.planning/STATE.md`
- **Config:** `.planning/config.json`

### Commands

- **Check progress:** `/gsd-progress`
- **Plan next phase:** `/gsd-plan-phase <N>`
- **Execute phase:** `/gsd-execute-phase <N>`
- **Verify work:** `/gsd-verify-work`

### Key Project Info

- **Core value:** Análise crítica sobre violência algorítmica em cidades inteligentes
- **Method:** Revisão sistemática da literatura
- **4 phases:** Estruturação → Revisão Sistemática → Análise → Redação
- **19 requirements** total
- **PICOC:** Population (populações marginalizadas), Intervention (cidades inteligentes/IA), Outcome (desigualdades), Context (áreas urbanas)
