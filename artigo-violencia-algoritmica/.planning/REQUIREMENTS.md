# Requirements: Violência Algorítmica em Cidades Inteligentes

**Defined:** 2026-05-14
**Core Value:** Produzir uma análise crítica e sistemática sobre violência algorítmica no contexto urbano, com rastreabilidade completa entre referências, dados extraídos, achados e conclusões.

## v1 Requirements

### Projeto e Estruturação (PRJ)

- [ ] **PRJ-01**: Estruturar diretórios do repositório (references, data, writing)
- [ ] **PRJ-02**: Definir string de busca sistemática a partir dos termos PICOC
- [ ] **PRJ-03**: Documentar protocolo de revisão com critérios de inclusão/exclusão
- [ ] **PRJ-04**: Configurar arquivo BibTeX mestre com todas as referências

### Revisão Sistemática (SLR)

- [ ] **SLR-01**: Triar artigos coletados por status (Accepted vs Rejected vs Duplicated)
- [ ] **SLR-02**: Extrair dados padronizados dos artigos aceitos (autor, ano, objetivo, método, achados)
- [ ] **SLR-03**: Sintetizar achados por questão de pesquisa
- [ ] **SLR-04**: Mapear lacunas na literatura identificadas

### Análise (ANA)

- [ ] **ANA-01**: Conceituar violência algorítmica e suas manifestações urbanas
- [ ] **ANA-02**: Identificar grupos sociais mais afetados por sistemas algorítmicos
- [ ] **ANA-03**: Discutir alternativas críticas propostas na literatura
- [ ] **ANA-04**: Analisar relações de poder no controle de dados urbanos

### Escrita (WRT)

- [ ] **WRT-01**: Redigir introdução com contextualização e objetivos
- [ ] **WRT-02**: Redigir referencial teórico sobre violência algorítmica
- [ ] **WRT-03**: Redigir metodologia (protocolo de revisão)
- [ ] **WRT-04**: Redigir resultados da revisão sistemática
- [ ] **WRT-05**: Redigir discussão e análise crítica
- [ ] **WRT-06**: Redigir conclusões
- [ ] **WRT-07**: Revisar e formatar artigo conforme normas ABNT/NBR

## v2 Requirements

### Dados Suplementares

- **SUPL-01**: Criar tabelas e figuras para visualização dos achados
- **SUPL-02**: Preparar material suplementar (planilha de extração completa)

## Out of Scope

| Feature | Reason |
|---------|--------|
| Coleta de dados primários | Escopo definido como revisão bibliográfica |
| Experimentos empíricos | Abordagem teórico-conceitual |
| Análise quantitativa | Abordagem qualitativa |
| Submissão a periódico | Escopo limitado à produção do artigo |
