# Roadmap: Violência Algorítmica em Cidades Inteligentes

**Milestone:** v1.0 — Artigo completo
**Início:** 2026-05-14

---

## Phase 1: Estruturação do Projeto e Referencial Teórico

**Goal:** Organizar o repositório, finalizar string de busca, documentar protocolo e consolidar referências bibliográficas.

**Requirements:**
- PRJ-01: Estruturar diretórios do repositório
- PRJ-02: Definir string de busca sistemática
- PRJ-03: Documentar protocolo de revisão
- PRJ-04: Configurar arquivo BibTeX mestre

**Success Criteria:**
1. Repositório estruturado com diretórios references/, data/, writing/
2. String de busca finalizada combinando termos PICOC com operadores booleanos
3. Protocolo de revisão documentado com critérios de inclusão/exclusão explícitos
4. Arquivo BibTeX contendo todos os artigos importados com metadados completos

---

## Phase 2: Revisão Sistemática da Literatura

**Goal:** Triar, extrair e sintetizar dados dos artigos aceitos, organizando achados por questão de pesquisa.

**Requirements:**
- SLR-01: Triar artigos por status (Accepted vs Rejected vs Duplicated)
- SLR-02: Extrair dados padronizados dos artigos aceitos
- SLR-03: Sintetizar achados por questão de pesquisa
- SLR-04: Mapear lacunas na literatura

**Success Criteria:**
1. Planilha de triagem completa com decisão final para cada artigo
2. Dados extraídos de todos os artigos Accepted em formato padronizado
3. Síntese textual organizada por cada uma das 4 questões de pesquisa
4. Documento de lacunas identificadas na literatura

---

## Phase 3: Análise e Discussão

**Goal:** Analisar criticamente a literatura para conceituar violência algorítmica, identificar grupos afetados, discutir alternativas e relações de poder.

**Requirements:**
- ANA-01: Conceituar violência algorítmica e suas manifestações urbanas
- ANA-02: Identificar grupos sociais mais afetados
- ANA-03: Discutir alternativas críticas
- ANA-04: Analisar relações de poder no controle de dados

**Success Criteria:**
1. Definição operacional de violência algorítmica no contexto urbano
2. Mapeamento dos grupos populacionais mais vulneráveis com evidências da literatura
3. Catálogo de alternativas críticas propostas (regulatórias, técnicas, sociais)
4. Análise das assimetrias de poder no ecossistema de dados urbanos

---

## Phase 4: Redação e Finalização do Artigo

**Goal:** Redigir, revisar e formatar o artigo acadêmico completo conforme normas.

**Requirements:**
- WRT-01 a WRT-07: Redação completa do artigo (introdução, teórico, método, resultados, discussão, conclusão, formatação)

**Success Criteria:**
1. Artigo completo com todas as seções redigidas
2. Referências formatadas corretamente (BibTeX)
3. Formatação ABNT/NBR aplicada
4. Revisão final de conteúdo e ortografia realizada
5. Documento exportado em formato final (.pdf/.docx)
