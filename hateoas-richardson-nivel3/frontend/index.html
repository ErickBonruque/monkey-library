Nível 3

Com HATEOAS ✨

Resposta inclui links dinâmicos indicando as ações possíveis baseadas no estado atual do recurso.

GET /contas/1 
{
  "id": 1,
  "titular": "Maria Silva",
  "saldo": 5200.00,
  "_links": {
    "sacar": { "href": "…/sacar" },
    "transferir": { "href": "…/transferir" },
    "extrato": { "href": "…/extrato" }
  }
}

✅ O front-end renderiza botões com base nos links recebidos!

Estes botões são gerados exclusivamente a partir dos links retornados pela API. O front-end não possui lógica própria para decidir o que exibir.

Como o Estado muda os Links?

O "S" de HATEOAS é State (Estado). Veja como o mesmo endpoint retorna links diferentes dependendo do estado.
